# Even and Odd Checker

## Description
A simple Java program that checks whether a user-entered number is **even** or **odd** using the modulus (`%`) operator.

## Features
- Accepts an integer as input.
- Determines whether the number is even or odd.
- Uses `Scanner` for user input.
- Displays the result on the console.

## Concepts Used
- Variables
- `Scanner` Class
- `if-else` Statement
- Modulus (`%`) Operator
