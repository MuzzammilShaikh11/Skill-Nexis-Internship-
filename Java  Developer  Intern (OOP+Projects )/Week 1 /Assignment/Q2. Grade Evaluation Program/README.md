# Grade Evaluation Program

## Description
This is a simple Java console application that evaluates a student's grade based on marks entered by the user. It uses conditional statements (`if-else`) to assign grades according to the score.

## Grade Criteria

| Marks | Grade |
|-------:|:-----:|
| 90–100 | A |
| 80–89  | B |
| 70–79  | C |
| 60–69  | D |
| Below 60 | F |

## Features
- Accepts marks as user input.
- Validates input (0–100).
- Displays the corresponding grade.
- Prints an error message for invalid marks.

## Concepts Used
- Java
- Scanner Class
- if-else Statements
- Input Validation
