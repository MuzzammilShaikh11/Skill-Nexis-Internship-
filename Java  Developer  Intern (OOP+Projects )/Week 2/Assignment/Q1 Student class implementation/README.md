# Student Class Implementation

## Objective
Create a Java class named `Student` with attributes and methods to input and display student information.

## Description
This program demonstrates the concept of **Classes and Objects** in Java. It allows the user to enter and display student details through a menu-driven interface.

## Student Attributes
- PRN
- Name
- CGPA
- Branch

## Features
- Create a `Student` class.
- Input student details.
- Display student information.
- Menu-driven program using `do-while` loop and `switch` statement.
- Exit option.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class

## Learning Outcomes
- Understand Java classes and objects.
- Use instance variables and methods.
- Take user input using the `Scanner` class.
- Implement a menu-driven program using loops and switch statements.
