# Vehicle Inheritance Example

## Objective
Implement inheritance in Java by creating a base class `Vehicle` and two subclasses `Car` and `Bike`. Demonstrate method overriding.

## Description
This program demonstrates the concept of **Inheritance** and **Method Overriding** in Java. The `Vehicle` class acts as the base class, while `Car` and `Bike` inherit from it and provide their own implementation of the `start()` method.
