# Question 1 – Student Class Using Constructor

## Objective
Create a `Student` class with the attributes `name`, `rollNo`, and `marks`. Read the student details using a constructor and display them using a method. Create the object from the `Question_1` class.

## Files

### Student.java
Contains the `Student` class with:
- Attributes:
  - `String name`
  - `int rollNo`
  - `int[] marks`
- Constructor to accept student details.
- `display()` method to print student information.

### Question_1.java
Contains the `main()` method that creates a `Student` object.

## Concepts Used
- Classes and Objects
- Constructors
- Arrays
- Methods
- Loops
- Scanner Class
- Object Creation
