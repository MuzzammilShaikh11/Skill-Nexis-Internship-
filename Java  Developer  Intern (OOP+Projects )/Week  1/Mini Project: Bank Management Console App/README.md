# Week 1 Mini Project

## Bank Management Console App

This is a simple Java console application developed as part of the Week 1 internship assignment. It simulates basic banking operations using a menu-driven interface.

### Features
- Check account balance
- Deposit money
- Withdraw money
- Exit the application

### Concepts Used
- Java
- Scanner
- Variables
- Loops
- Switch Case
- Conditional Statements
